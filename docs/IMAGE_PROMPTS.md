# Light Brand Consulting - AI Image Generation Prompts

This document contains carefully crafted AI image generation prompts for every image location in the Light Brand Consulting platform. Use these prompts with Midjourney, DALL-E, Stable Diffusion, or similar AI image generators.

**Brand Guidelines:**
- Primary Colors: Gold (#E8B84A), Amber (#D4944C), Cream (#FDF6E3)
- Background: Deep charcoal/slate (#0F0F10, #1A1A1C)
- Style: Premium, sophisticated, warm illumination themes
- Avoid: Generic stock photo looks, overly corporate, cold tech aesthetics

## Usage Notes

1. **Consistency is Key**: When generating, keep the same style and color palette across all images
2. **Test on Dark Backgrounds**: All images should look great on the dark charcoal (#0F0F10) base
3. **Optimization**: Compress all images for web (target <200KB for large images, <50KB for icons)
4. **Retina Ready**: Generate at 2x for hero images to support high-DPI displays
5. **Accessibility**: Ensure sufficient contrast between image and overlaid text
6. **Brand Alignment**: When in doubt, lean toward warm, sophisticated, and premium over cold, techy, or generic
